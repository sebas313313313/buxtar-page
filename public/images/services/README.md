# Carpeta de Imágenes de Servicios

Aquí puedes colocar las imágenes reales para cada sección de los servicios.
Por ahora, la página usa imágenes de prueba (placeholders) desde internet, pero puedes reemplazarlas apuntando la ruta en el código hacia `/images/services/tu-imagen.jpg`.

Se recomienda usar:
- Imágenes rectangulares o cuadradas (ej. 800x600 px).
- Formatos optimizados como .webp, .jpg o .png.
